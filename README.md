# macapture_support

Support website for [macapture](https://apps.apple.com), a screen capture utility for macOS.

## Pages

- **`index.html`** — Main support page: Getting Started guide, FAQ accordion, Troubleshooting tips, and a contact form.
- **`privacy.html`** — Privacy Policy.
- **`style.css`** — Shared stylesheet.

## Hosting

This site is designed to be hosted via [GitHub Pages](https://pages.github.com/). To enable it, go to **Settings → Pages** in this repository and set the source to the `main` branch.
